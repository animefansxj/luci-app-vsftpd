luci-app-vsftpd
=================================================
VERSION 5.0

简述:
    该软件包用于向OpenWRT提供一个基于LuCI的VSFTPD
控制页面，该页面允许对VSFTPD进行常规参数以及用户
控制的设置。


测试环境:
    在 OpenWRT 15.05.1 上开发并测试通过


依赖关系:
    vsftpd libpam libdb47 db47
